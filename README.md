# Онлайн-запись в велоремонт

**Студент:** Утин М.А., группа 454  
**СУБД:** MySQL 8.0  
**Описание:** Система для записи клиентов на ремонт велосипедов с учётом простых/сложных услуг, опыта мастеров, аналогов запчастей.

## Структура репозитория
- `sql/schema.sql` – полный скрипт создания БД, таблиц, ограничений и тестовых данных.
- `answer_question_23_variant_23.md` – ответ на контрольный вопрос №23.
- `docs/report.pdf` – отчёт по практической работе (все разделы).
- `docs/images/` – скриншоты выполнения запросов.

## Как развернуть
1. Клонировать репозиторий.
2. Выполнить в MySQL: `source sql/schema.sql`.
3. Запросы из отчёта можно выполнять в любом клиенте.
